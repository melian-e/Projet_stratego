Règles :

    Stratego est un jeu de stratégie dont le but est de capturer le drapeau ennemi.
    Les joueurs doivent respecter plusieurs règles :
    - Chaque joueur peut déplacer un de ses pions d’une case par tour.
    - Lors d’une attaque, la plus faible puissance meurt, si les deux sont égaux, les deux pions meurent.
    - Aucune pièce ne peut bouger entre deux déplacements pour trois tours.
    - Interdiction de pourchasser une pièce qui s’enfuit.
    - Si le drapeau ou toutes les pièces déplaçables d’un joueur sont capturés, victoire de l’adversaire.
    Certaines règles sont choisies par les joueurs en début de partie.
